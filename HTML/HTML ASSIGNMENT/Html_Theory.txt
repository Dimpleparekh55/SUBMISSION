1.	Are the HTML tags and elements the same thing?
Ans. HTML tags and elements are different things. HTML tags are consists of the element name within angle brackets (<>), whereas elements are identified by tags in text source.

• What are tags and attributes in HTML?
Ans. Attributes are instructions that clarify or modify an element and provides the location of the image file via its URL. Tags are the way of representing an HTML element in the program.
 
• What are void elements in HTML?
Ans. A void element is an element whose content model never allows it to have contents under any circumstances.

• What are HTML Entities?
Ans. A commonly used entity in HTML is the non-breaking space: &nbsp; A non-breaking space is a space that will not break into a new line.
 
• What are different types of lists in HTML?
Ans. There are four types of lists in HTML. They are Order list, Unorder list, Nested list and Definition list. 

• What is the ‘class’ attribute in HTML? 
Ans. The HTML class attribute is used to specify a class for an HTML element.
Multiple HTML elements can share the same class. 

• What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements? 
Ans. The difference between id and class attribute is the only that “id” is unique in a page and can only apply to at most one element, while “class” selector can apply to multiple elements.

• What are the various formatting tags in HTML? 
Ans. In HTML, a number of elements are used to format text. The <h1>-<h6> tags are used to define HTML headings. There are 6 levels of headings in HTML, the <h1> defines the most and the <h6> least important headings. The <b> tag is a physical tag that stands for bold text. The <p> tag defines the paragraph. Browsers automatically add 1em margin before and after each paragraph. The <br> tag inserts a single line break. Unlike the <p> tag an empty indent is not added before the line. In HTML5 the <hr> tag defines a thematic change between paragraph level elements in an HTML page. In previous versions of HTML it was used to draw a horizontal line on the page visually separating the content. In HTML5 the element has semantic meaning.

• How is Cell Padding different from Cell Spacing? 
Ans. Cell padding is the amount of space held between the contents of the cell and the cell border, whereas Cell spacing is the amount of space held between cells, specified in number of pixels.

• How can we club two or more rows or columns into a single row or column in an HTML table?
 Ans. We can club two or more columns into a single column using the colspan attribute in a <td> tag and we can club two or more rows into a single row using the rowspan attribute in a <td> tag.

• What is the difference between a block-level element and an inline element? 
Ans. The distinction between block-level and inline elements is that if the heading and paragraph elements start on new lines and do not run together on same line is block-level element and an inline element does not start a new line, but rather
stays in the flow of the paragraph.

• How to create a Hyperlink in HTML? 
Ans. To create a hyperlink, you should use the <a> tag and href attribute, the value of which is the URL, or location, where the link is pointing to. Create a hyperlink using the id of the link target, preceded by hash (# )
Ex. <a href="#jump">When you click on this link, you will be redirected to the part of the page with "jump" id <a>.

• What is the use of an iframe tag? 
Ans. The <iframe> tag creates an inline frame for embedding third-party content (media, applets, etc.). Although the content of the frame and the web page are independent, they can interact through JavaScript.

• What is the use of a span tag? Explain with example? 
Ans. The <span> tag is a generic inline container that groups elements primarily for styling purposes. The <span> tag, as well as <div> tag are empty containers. The <div> tag groups block level elements, whereas the <span> groups inline elements. The content hooked in the <span> element, can be styled with CSS, or manipulated with JavaScript. 
Here are some examples of the HTML <span> tag usage:

Highlight a part of a text with color, define a background color or background image for it.
Change the font of separate words or phrases.
Apply scripts to certain parts of the text, e.g., highlight the code syntax.

• How to insert a picture into a background image of a web page? 
Ans. In the body tag, specify a background image in the background attribute by passing the URL of the image or location path.

• How are active links different from normal links? 
Ans. Normal links are links which are there on the page and have not been clicked yet. Active links are those links, which have just been clicked at that instant.

• What are the different tags to separate sections of text? 
Ans. <br> tag – Usually <br> tag is used to separate the line of text. It breaks the current line and conveys the flow to the next line. <p> tag – This contains the text in the form of a new paragraph.

• What is SVG? 
Ans. SVG (Scalable Vector Graphics) is a language for two-dimensional graphics based on XML with support for animation and interactivity. To draw images, it uses simple geometrical figures (circle, lines, polygon, etc).

• What is difference between HTML and XHTML? 
Ans. HTML is the standard markup language for creating web pages, while XHTML is a stricter and more standardized version of HTML. Both HTML and XHTML include a wide range of features, such as support for multimedia, styling, and scripting.

• What are logical and physical tags in HTML? 
Ans. physical tags, that are used to style the text (visual appearance of the text) and logical or semantic tags that add semantic value to the text parts (e. g., inform search engines for which keywords a web page should be ranked).
